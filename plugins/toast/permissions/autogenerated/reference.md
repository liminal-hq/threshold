## Default Permission

Default permissions for the toast plugin

#### This default permission set includes the following:

- `allow-show`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>

</td>
<td>

</td>
</tr>

<tr>
<td>

`toast:deny-show`

</td>
<td>

</td>
</tr>
