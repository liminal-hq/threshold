# UI Screen Specs

This folder contains baseline screen specifications for Threshold UI work.

## Purpose

- capture current in-code behaviour before redesign work
- provide stable UI contracts for future rework
- separate visual design direction from functional expectations

## Baseline Date

- 2026-03-08

## Usage Notes

- update each screen spec when behaviour changes, not only when visuals change
- model platform differences explicitly using:
  - `Platform Concept Model` (Shared, Mobile, Desktop)
  - `Platform Mapping Matrix`
- include state handling and interaction rules, not just layout notes
